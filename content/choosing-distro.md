# Choosing a Distro

A **"distro"** (short for *distribution*) is a version of Linux packaged with a desktop environment and a set of default apps. Think of it like different flavours of ice cream — they're all ice cream, but they taste a little different! 🍦

For Windows users, we recommend starting with one of these two beginner-friendly options:

---

## 🟠 Ubuntu

**Best for:** First-timers who want the most support and resources

Ubuntu is the most popular Linux distro in the world. It's backed by a company called Canonical and has an enormous community, meaning you'll almost always find an answer to your question online.

### What makes Ubuntu great for Windows switchers:
- **Familiar layout** — the taskbar, app launcher, and system tray will feel recognizable
- **Huge software library** — almost every Linux app supports Ubuntu first
- **Long-term support (LTS)** — stable versions supported for 5 years
- **Massive community** — millions of tutorials, forum posts, and YouTube videos

### What to expect:
- Uses the **GNOME** desktop — clean and modern
- Ships with Firefox, LibreOffice, and the Ubuntu App Center
- Free upgrades to new versions every 2 years (LTS) or 6 months

> 🔗 **Download:** [ubuntu.com/download/desktop](https://ubuntu.com/download/desktop)

---

## 🔵 Pop!_OS

**Best for:** Users who want a polished, modern experience — especially great for gaming and creative work

Pop!_OS is made by System76, a company that builds Linux laptops. It's based on Ubuntu (so all Ubuntu software works on it!), but with a more refined, distraction-free interface.

### What makes Pop!_OS great for Windows switchers:
- **Beautiful, clean design** — feels like a premium operating system
- **Excellent hardware support** — especially for NVIDIA graphics cards
- **Auto-tiling** — windows snap and arrange automatically (you can turn this off)
- **Recovery partition** — easy system recovery if something goes wrong

### What to expect:
- Uses a customised **GNOME** desktop — more polished than stock Ubuntu
- Ships with Pop!_Shop for easy app installation
- Special ISO available for NVIDIA GPU users

> 🔗 **Download:** [pop.system76.com](https://pop.system76.com)

## 🤔 Which Should YOU Choose?

**Choose Ubuntu if you:**
- Want maximum community support and tutorials
- Are using an older computer
- Just want to get started with the most popular option

**Choose Pop!_OS if you:**
- Have an NVIDIA graphics card
- Want a more polished out-of-the-box experience
- Plan to use your computer for gaming or creative work
- Like a cleaner, more focused desktop

> 💡 **Remember:** You can always try both! Use a USB drive to test each one before installing. See the [Installation Guide](#installation) for how to do this.
