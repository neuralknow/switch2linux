# Recommended Apps

One of the first things to do after installing Linux is setting up your favourite apps. Good news — most of what you need is either available natively on Linux, or has an excellent alternative! 🛍️

---

## How to Install Apps on Linux

There are several ways to install apps:

| Method                         | Best for                | How                                           |
| ------------------------------ | ----------------------- | --------------------------------------------- |
| **App Centre / Software Shop** | Most users              | Open the app, search, click Install           |
| **Terminal (apt)**             | Quick installs          | `sudo apt install appname`                    |
| **Flatpak**                    | Universal apps          | Via App Centre or `flatpak install`           |
| **Snap**                       | Ubuntu apps             | `sudo snap install appname`                   |
| **Direct download**            | Some apps (e.g. Chrome) | Download `.deb` file, double-click to install |

## Utilities & System Tools

| App           | Windows equivalent   | Notes                                            |
| ------------- | -------------------- | ------------------------------------------------ |
| **Timeshift** | System Restore       | Automated system snapshots — install this first! |
| **BleachBit** | CCleaner             | System cleaner                                   |
| **KeePassXC** | LastPass / 1Password | Local password manager                           |
| **Bitwarden** | LastPass             | Cloud password manager — same app as Windows     |
| **Flameshot** | Snipping Tool        | Powerful screenshot tool                         |
| **CopyQ**     | Clipboard managers   | Clipboard history manager                        |
| **Syncthing** | OneDrive / Dropbox   | P2P file sync across devices                     |
> 💡 **Start with Timeshift!** Before making major changes to your system, set up Timeshift to take a snapshot. If anything goes wrong, you can restore your system to a known-good state in minutes.

## Finding More Apps

- **App Centre / Pop!_Shop** — browse categories like a mobile app store
- **Flathub** — [flathub.org](https://flathub.org) — the largest collection of Flatpak apps
- **Snapcraft** — [snapcraft.io](https://snapcraft.io) — Snap packages from Canonical
- **AlternativeTo** — [alternativeto.net](https://alternativeto.net) — find Linux alternatives to any Windows app
